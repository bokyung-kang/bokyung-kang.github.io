스프링 배치 Job Runner를 조사하던 중에 Jenkins에서 gradle빌드해 보았습니다. Jenkins 환경 Amazon Linux2 AMI ID : ami-0f27d081df46f326c AMI 이름 : amzn2-ami-hvm-2.0.20210303.0-x86_64-gp2 Jenkins version : Jenkins 2.284 인증정보 등록 Github 아이디/패스워드 인증 방법 Jenkins에서 Repository관련 작업이 가능하도록 Github에서 Personal access tokens를 발행합니다.
